Getting started:
1. install base tools:
    node.js,
    cygwin with git (needed for bower),
    a javascript oriented IDE (WebStorm is a good one)
2. install node based tools:
    npm install -g gulp
    npm install -g bower
3. fetch project dependencies:
    npm install && bower install
4. build the project:
   gulp

5. running the test server:
   node server/superstatic.js
